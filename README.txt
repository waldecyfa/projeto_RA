Projeto Prático de Realidade Aumentada
# A-Frame AR Project

## Como usar

1) Baixe um modelo .glb do Sketchfab
2) Renomeie para claw.glb
3) Coloque em /assets
4) Abra index.html via localhost ou HTTPS
5) Use o marcador HIRO do AR.js

## Interações

Segurar clique/toque -> aumenta escala  
Toque rápido -> efeito pulso
